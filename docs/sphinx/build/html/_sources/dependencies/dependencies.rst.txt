.. _dependencies:

#####################
External dependencies
#####################

Although the nanodiffraction toolbox is almost stand-alone, it uses one dependency that might not be available to the user. This external dependency cannot unfortunately be offered by the authors for licensing reasons. This concerns the following function:
    ``read_pilatus``
    
Other external dependencies include the three colormap packages mentioned in :ref:`colormaps`. The given licenses allow for distribution of the software within this package as long as it is accompanied by the original license.
