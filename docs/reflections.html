Literature values for lattice spacings - IRP edition

The following list summarizes the lattice spacings commonly encountered at the IRP. The list is non-exhaustive and the authors would like to invite everyone to extend this list. In order to do so, please register with Github, login to your account and become a collaborator of the Github account irpgoe. You can then directly edit this site using the tools appearing on the right side of this list on hovering with the mouse cursor over an entry in the list. Please report any bugs or errors to the authors or directly correct the mistake. Thank you!
